# Dental Clinic Inventory App (v3)

A React + Google Sheets inventory system for managing multi-location dental stock.

## Features
- **Warehouse:** Search inventory, add to cart, generate QR code for transfer.
- **Clinic:** Scan QR code to receive stock (updates Google Sheet instantly).
- **Tech Stack:** React, TailwindCSS, Google Apps Script.
